
# LibScratch

Short scratch for setting up your own python development library. Use it as the simplest template.

## Project UML

Link to your UML (if needed).

## Dependencies

See requirements.txt for details.

## Installation guide

To install, run right here

```apache
pip install -e .
```
and the library will be installed in editable mode, i.e. env will store only link to it so that all modifications will not require re-installation.

## Quick Run Examples

If you want some, place it here.

```python
import numpy as np
import libscratch.exampleModule as ex

a = np.array([1,2])
b = np.array([3,4])
print(ex.customInnerProduct(a,b))
```
